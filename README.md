# SMART ONLINE SURVEY SYSTEM WITH ANALYTICS

A dynamic web-based survey platform developed using **HTML, CSS, JavaScript, Firebase Authentication, and Firebase Realtime Database**.

The system enables users to create surveys, collect responses, and analyze feedback through a modern and responsive interface.

## Features

* Secure User Registration and Login
* Dynamic Survey Creation
* Multiple Question Types (Text, Multiple Choice, Rating, Yes/No)
* Real-Time Response Collection
* Interactive Dashboard
* Analytics and Data Visualization
* Responsive and Professional UI

## Benefits

* Reduces paperwork
* Saves time in data collection
* Secure cloud-based storage
* Easy survey management
* Real-time analytics and insights
* Improves decision-making process

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Firebase Authentication
* Firebase Realtime Database
* Chart.js
